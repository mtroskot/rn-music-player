# rn-music-player

Music player for playing music from iOS's and Android's Music Library

## Installation

```sh
npm install rn-music-player
```

## Usage

```js
import RnMusicPlayer from "rn-music-player";

// ...

const result = await RnMusicPlayer.multiply(3, 7);
```
